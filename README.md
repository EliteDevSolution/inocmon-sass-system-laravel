# laravel-soccer-sportmonks-api

## Get Started

env setting

Composer install

```sh
$ composer install
```

Laravel license

```sh
$ php artisan key:generate
```

Database migrate and seed

```sh
$ php artisan vendor:publish --tag="cashier-migrations"
$ php artisan migrate --seed
```

Storage link

```sh
$ php artisan storage:link
```

Run serve

```sh
$ php artisan serve
```
